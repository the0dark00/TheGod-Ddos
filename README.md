# TheGod-Ddos
This is a DDoS control panel with extremely easy-to-use features; you only need to enter the website URL and the number of packets to be delivered to the site. 






Prerequisites:
Python 3.6+
pip (Python package manager)

1 Clone the Repository
git clone https://github.com/the0dark00/TheGod-Ddos.git

cd TheGod-Ddos

2 Install Requirements
All dependencies are listed in requirements.txt.
pip install -r requirements.txt
Step-by-Step Usage Instructions


Termux:

1 pkg update && pkg upgrade

2 pkg install git python

3 git clone https://github.com/the0dark00/TheGod-Ddos.git

4 cd TheGod-Ddos

5 pip install -r requirements.txt

6 python3 start.py



Debian-based GNU/Linux:

1 sudo apt update && sudo apt upgrade

2 sudo apt install git python3 python3-pip

3 git clone https://github.com/the0dark00/TheGod-Ddos.git

4 cd TheGod-Ddos

5 pip3 install -r requirements.txt

6 python3 start.py



Windows:

1 Download & install Python.

2 Open cmd or PowerShell and run:

3 git clone https://github.com/the0dark00/TheGod-Ddos.git

4 cd TheGod-Ddos

5 pip install -r requirements.txt

6 python start.py



MacOS:

1 brew install git python3

2 git clone https://github.com/the0dark00/TheGod-Ddos.git

3 cd thegod
