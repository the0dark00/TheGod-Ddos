# TheGod-Ddos
This is a DDoS control panel with extremely easy-to-use features; you only need to enter the website URL and the number of packets to be delivered to the site. 






Prerequisites
Python 3.6+
pip (Python package manager)
1. Clone the Repository
git clone https://github.com/the0dark00/TheGod-Ddos.git
cd thegod
2. Install Requirements
All dependencies are listed in requirements.txt.

pip install -r requirements.txt
Step-by-Step Usage Instructions
Termux
pkg update && pkg upgrade
pkg install git python
git clone https://github.com/the0dark00/TheGod-Ddos.git
cd thegod
pip install -r requirements.txt
python3 dos.py
Debian-based GNU/Linux
sudo apt update && sudo apt upgrade
sudo apt install git python3 python3-pip
git clone https://github.com/the0dark00/TheGod-Ddos.git
cd thegod 
pip3 install -r requirements.txt
python3 dos.py
Windows
Download & install Python.
Open cmd or PowerShell and run:
git clone https://github.com/the0dark00/TheGod-Ddos.git
cd thegod
pip install -r requirements.txt
python dos.py
MacOS
brew install git python3
git clone https://github.com/the0dark00/TheGod-Ddos.git
cd thegod
pip3 install -r requirements.txt
python3 dos.py
Usage
After installation, run the main file:

python3 dos.py
or (on Windows):

python dos.py
